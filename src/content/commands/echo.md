# echo

## 概要

- 文字列の出力と、コマンドエコー（表示）制御を行います。

## 代表的な使い方

```cmd
echo Hello
```

```cmd
@echo off
echo ログを開始します
```

## オプションを使用した例

```cmd
echo.
```

## オプション / 引数

| 要素 | 説明 |
| --- | --- |
| `on` / `off` | コマンド表示を切り替える |
| `.` | 空行を出力する |

## よくある落とし穴

- `echo.` は環境によって解釈差があるため、必要なら `echo(` も検討する
- 先頭/末尾空白は見落としやすい
