# active

## 概要

- Windows の **cmd.exe に組み込まれている単体コマンドとしての `active`** は、一般的なリファレンスでは扱われません。製品付属ツールやスクリプト内のサブコマンド名として現れることはあります。

## 代表的な使い方

```cmd
REM cmd 標準に「active」単体の定型例はないため、環境固有のヘルプを参照する
where active
```

```cmd
REM 同名の実行ファイルがある場合のみ
active /?
```

## オプションを使用した例

```cmd
REM ツールによって全く異なるため、利用中の製品ドキュメントを参照する
```

## オプション / 引数

| 要素 | 説明 |
| --- | --- |
| （環境依存） | 実行ファイルが存在する場合のみ、そのツールの仕様に従う |
