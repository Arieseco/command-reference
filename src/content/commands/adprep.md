# adprep

## 概要

- **Active Directory ドメインサービス（AD DS）** のアップグレードや新規ドメインコントローラ追加の前後に、スキーマや権限の準備を行うためのツールです。通常は **ドメイン管理者** が、ドメインコントローラ上（または手順に従った環境）で実行します。一般向けの日々の cmd 作業では使いません。

## 代表的な使い方

```cmd
REM 例：フォレスト準備（バージョンは環境・手順書に合わせる）
adprep /forestprep
```

```cmd
REM 例：ドメイン準備
adprep /domainprep
```

## オプションを使用した例

```cmd
REM 例：ドメインコントローラを追加するドメインの準備（手順に依存）
adprep /domainprep /gpprep
```

```cmd
REM ヘルプ表示（実装・バージョンで表記が異なる場合あり）
adprep /?
```

## オプション / 引数

| 要素 | 説明 |
| --- | --- |
| `/forestprep` | フォレスト全体の準備（要件・順序は厳密） |
| `/domainprep` | ドメインの準備 |
| `/rodcprep` | RODC 関連の準備（利用する場合） |

※ 実際のオプションは **Windows Server のバージョンとマイクロソフトの手順書** を必ず参照してください。

## よくある落とし穴

- **誤実行するとドメイン全体に影響** するため、テスト環境と手順のダブルチェックが必須
- 実行ファイルの場所はインストールメディアや `support\adprep` など **環境により異なる**
- 本サイトでは **先頭文字 A の一覧試験** 用にもエントリを置いている
