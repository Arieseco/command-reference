# bdehdcfg

## 概要

- **BitLocker ドライブ暗号化** を有効にする前に、必要な **ディスク構成（システムパーティションの作成など）** を行うツールです。古い構成の PC で TPM と連携した暗号化を進めるときなどに、管理者権限で実行します。

## 代表的な使い方

```cmd
REM ヘルプを表示
bdehdcfg -?
```

```cmd
REM 対話的に構成を変更（環境によりメッセージが異なる）
bdehdcfg
```

## オプションを使用した例

```cmd
REM 新規システムパーティションを作成する例（ディスク構成による）
bdehdcfg -target default -size 300
```

## オプション / 引数

| 要素 | 説明 |
| --- | --- |
| `-target` | 対象（例: `default`）や縮小元ボリュームの指定 |
| `-size` | 作成するパーティションのサイズ（MB） |
| `-quiet` | 対話を減らす |
| `-restart` | 必要に応じて再起動を促す |

※ OS バージョンにより **利用できるスイッチが異なる** 場合があります。`/help` や `-?` で確認してください。

## よくある落とし穴

- **ディスクの再パーティション** を伴うため、事前のバックアップが重要
- 仮想マシンや特殊なディスクでは **想定外のメッセージ** になることがある
