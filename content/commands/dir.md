# dir

## 概要

- カレントまたは指定パスのファイル一覧を表示します。

## 代表的な使い方

```cmd
dir
```

```cmd
dir C:\Windows
```

## オプションを使用した例

```cmd
dir /b /a
```

## オプション / 引数

| 要素 | 説明 |
| --- | --- |
| `/b` | 簡易表示（名前のみ） |
| `/a` | 属性指定で表示対象を変える |
| `/s` | サブディレクトリも含める |

## よくある落とし穴

- `/s` は結果が大量になりやすい
- 出力を後で処理する場合は `/b` を使うと扱いやすい
